Rangkuman Pemrogaman Mobile

GitHub

1. Instalisasi GitHub dengan mengetik peritah git init. Perintah terebut untuk membuat repository local untuk sebuah projek

2. Lalu untuk memeriksa status projek yang kita buat kita mengetik perintah git status

3. Untuk menambahkan sebuah file source code beserta file projek kedalam staging area ketik perintah git add (nama file)

4. Setelah itu untuk menyimpan source code kedalam repository ketik perintah git commit –m “(perintah)”

5. Untuk menambah seluruh file dengan tipe yang sama ketik git add ‘*.(tipe file)’

6. Untuk melihat apa saja yang telah dirubah sejauh yang telah dibuat ketik perintah git log

7. Untuk menambah remote name dan repository URL agar projek kita bisa diupload maka ketik perintah git remote add origin https://github.com/ (nama owner)/(nama repository).git

8. Untuk mengupload ke GitHub ketik perintah git push –u origin master. –u berfungsi untuk memberitahu git supaya dia mengingat parameter. Sehingga kedepannya kita hanya perlu menjalankan git push dang it akan mengetahui apa yang harus dilakukan.

9. Untuk mengecek dan memperbarui GitHub yang telh dibuat maka ketik perintah git pull origin master

10. Untuk melihat perubahan yang belum masuk ke area stage maka ketik git diff (nama file)

11. Untuk melihat perubahan yang baru saja dimasukan kedalam area stage maka ketik perintah git diff –staged

12. Untuk menghilangkan perubahan dari commit yang telah terjadi maka ketik perintah git reset (namafile)

13. Untuk mengembalikan perubahan didalam projek maka gunakan perintah git checkout –(nama fie)

14. Untuk membuat branch (atau copy) pada source code maka bisa menggunakan perintah git branch (nama branch)

15. Untuk beralih ke branch yang telah dibuat maka bisa menggunakan perintah git checkout (nama branch)

16. Untuk menghapus file bisa menggunakan git rm ‘(nama file).(jenis file)’ atau bisa menggunakan git rm’*.(jenis file)’ untuk menghapus semua file dengan jenis yang telah diketik

17. Untuk kembali ke branch utama atau master branch maka bisa menggunakan perintah git checkout master

18. Untuk menghapus branch yang sudah dibuat bisa menggunakan perintah git merge –d (nama branch)

19. Untuk mengupload semua projek yang sudah dibuat maka bisa mengetik perintah git push
